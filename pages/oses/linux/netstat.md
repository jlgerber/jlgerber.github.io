### Netstat

#### Command Flags

##### List All Ports (listening and non-listening)
```
netstat -a
```

##### List All tcp ports
```
netstat -at
```

##### List All udp ports
```
netstat -au
```

##### List All Sockets in a Listening State
```
netstat -l
```

##### List All TCP Ports
```
netstat -lt
```

##### List All UDP Ports
```
netstat -lu
```

##### List listening UNIX Ports
```
netstat -lx
```

##### Show Statistics for protocols
```
netstat -s
```

##### Display PID and program names
add the ```-p``` flag.
```
netstat -pt
```

##### show list of network interfaces
```
netstat -i
```

##### Show extending list of network interfaces
```
netstat -ie
```
